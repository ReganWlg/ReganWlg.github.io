hexo博客创建、部署到本地及云端

Posted on 2020-10-07

创建

hexo n "博客名"
或者 hexo new "博客名"

部署到本地

hexo clean
这句可以没有

hexo g
或者 hexo generate

本地查看

hexo s
或者 hexo server

部署到云端

hexo d
或者 hexo deploy
